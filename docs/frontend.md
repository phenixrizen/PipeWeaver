# Frontend structure

The frontend uses Vue 3, Vite, TypeScript, Tailwind CSS, Vue Router, and Pinia.

## Views

- `PipelineListView.vue`: dashboard listing stored pipelines.
- `PipelineEditorView.vue`: editor and preview workspace.

## Components

- Layout shell and sidebar navigation.
- Connector config cards and format selectors.
- Mapping table with inline transform editing.
- Sample payload editor and output preview panels.
